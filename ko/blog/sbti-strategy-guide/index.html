이 가이드 사용법

  1. 먼저 원하는 성격 유형을 정한 뒤 아래에서 찾습니다.
  2. 테스트를 열고 “테스트 시작”을 누릅니다.
  3. 문항 순서는 매번 섞이므로, 문장 내용으로 어떤 문제인지 확인하세요.
  4. 원하는 성격의 답안 카드에서 그 문항을 찾고 “선택” 열에 적힌 글자대로 누르면 됩니다.
  5. 선택지는 A / B / C 입니다. A = 첫 번째, B = 두 번째, C = 세 번째입니다.
  6. 일반 문항 30개와 특수 문항 1개를 모두 끝내면 “제출하고 결과 보기”를 누르세요.
