¿Qué es el test SBTI?

El test de personalidad SBTI (Silly Big Personality Test) es una parodia humorística del MBTI. A diferencia del MBTI tradicional, el test SBTI es puro humor y no tiene base científica. Preguntas aleatorias, resultados fáciles de compartir.

¿Cómo se relaciona SBTI con MBTI?

SBTI es la versión divertida y paródica de MBTI. Si crees que los tests MBTI son demasiado serios, SBTI te da la misma experiencia con resultados más absurdos. SBTI es la alternativa divertida al MBTI.

¿Por qué hacer el test SBTI?

El test SBTI es completamente gratis y sin registro. Los resultados son perfectos para compartir con amigos.

¿Para quién es el test SBTI?

Para cualquier persona que busque un test de personalidad ligero y divertido. Ya seas veterano del MBTI o principiante, SBTI te da un resultado divertido.

FAQ

Preguntas frecuentes

Que es SBTI?

SBTI es un test de personalidad humoristico, pensado para bromear, reirse de uno mismo y compartir resultados.

Es preciso el resultado SBTI?

El resultado de SBTI puede sentirse muy certero, pero sigue siendo un test de personalidad hecho para el humor. No es una evaluacion clinica ni una herramienta seria.

Cuanto tiempo toma SBTI?

SBTI suele tomar lo que dura un cafe. Este test de personalidad no tiene limite de tiempo, asi que puedes responder por instinto.

Por que cambia el orden de preguntas de SBTI?

SBTI cambia el orden de preguntas en cada intento y a veces muestra preguntas especiales. Eso hace que cada test se sienta un poco distinto.

Que pasa si mi resultado SBTI es terrible?

Si tu resultado de SBTI sale fatal, tambien forma parte de la broma. Este test esta hecho para entretener y compartir.

Por que SBTI tiene tipos ocultos?

Algunas rutas de respuesta de SBTI activan resultados ocultos o de reserva. Eso hace que el test tenga mas gracia y rejugabilidad.
