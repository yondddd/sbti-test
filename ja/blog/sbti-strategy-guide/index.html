SBTI攻略ガイド：27タイプの回答カードとDRUNK / HHHHの出し方

27タイプ分の回答カードに加え、隠しタイプDRUNKの発動条件とHHHHになる条件までまとめたSBTI攻略ガイドです。

このガイドの使い方

  1. まず欲しいタイプを決めて、下の一覧から探します。
  2. テストを開いて「テスト開始」を押します。
  3. 問題の順番は毎回ランダムなので、文面でどの問題かを見分けてください。
  4. 狙っているタイプの回答カードでその問題を見つけ、「選択」列の文字どおりに押します。
  5. 選択肢は A / B / C です。A = 1番目、B = 2番目、C = 3番目です。
  6. 通常30問と特別問題1問をすべて答えたら、「送信して結果を見る」を押します。

重要: テスト中には趣味についての特別問題が4択でランダムに出ます。DRUNKを狙わないなら「飲酒」は絶対に選ばないでください。他の3つならどれでも構いません。

これは隠しタイプです。通常30問をどう答えても、特別条件を満たした瞬間に通常判定はすべて上書きされ、結果は強制的にDRUNKになります。

発動方法（2ステップだけ）

  1. 問題「普段の趣味は何ですか？」が出たら、D「飲酒」を選びます。
  2. 続けて追加問題「お酒に対するあなたの姿勢は？」が表示されるので、B「私は魔法瓶のカップにお酒を注ぎ、沸騰したお湯として飲むことに慣れています。お酒で納得です。」を選びます。

残りの通常30問は何を選んでも構いません。結果には一切影響しません。

HHHH(バカ笑い者) — フォールバックタイプ

はははははは。

これはシステムのフォールバックタイプです。回答パターンが25種すべての通常タイプに対して一致率60%未満だった場合、結果は強制的にHHHHへ振り分けられます。

発動方法

固定の回答式はありません。できるだけどのタイプにも噛み合わない答え方にする必要があります。

おすすめ戦略

  • 同じ次元に属する2問では、わざと逆の答えを入れてください。例えば片方をA、もう片方をCにすると、次元スコアが中間に落ちて特定のプロトタイプに寄りにくくなります。
  • さらに、次元どうしのパターンも互いに矛盾するようにしておくと安全です。
  • 趣味の特別問題では「飲酒」を選ばないでください。選ぶとHHHHではなくDRUNKが発動します。
  • このタイプは運にも左右されるので、何回かやり直しが必要になることがあります。
