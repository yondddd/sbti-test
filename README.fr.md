## Apercu

SBTI Test est un test de personnalite SBTI pense pour le divertissement. Ce depot contient la version statique multilingue du site, tandis que `README.md` reste le document principal du projet.

## Qu'est-ce que SBTI Test ?

SBTI signifie `Silly Big Personality Test`. Le test de personnalite SBTI garde le parcours familier ou l'on repond a des questions puis recoit un resultat, mais son ton reste humoristique, satirique et facile a partager.

- SBTI est gratuit
- le test de personnalite SBTI est fait pour s'amuser
- les resultats sont concus pour etre partages
- ce n'est pas un outil scientifique, medical ou clinique

## Relation entre SBTI et MBTI

SBTI se presente comme une parodie legere de MBTI. Il reprend le format connu du test MBTI, sans chercher a devenir un cadre serieux de personnalite.

## Les 15 dimensions du test de personnalite SBTI

Le test de personnalite SBTI utilise 15 dimensions regroupees en cinq blocs :

- modele de soi
- modele emotionnel
- modele d'attitude
- modele de motivation a l'action
- modele social

Le resultat final vient donc d'une lecture organisee sous plusieurs angles, pas d'une seule blague rapide.

## FAQ

### Qu'est-ce que SBTI ?

SBTI est un test de personnalite SBTI humoristique, concu pour donner des resultats rapides et faciles a partager.

### En quoi SBTI est-il different de MBTI ?

MBTI est le point de comparaison connu ; SBTI en est une version parodique et legere. Le plaisir compte plus que la precision scientifique.

### Pourquoi SBTI a-t-il 15 dimensions ?

Parce que le projet organise le resultat selon plusieurs angles avant d'afficher le type final. Cela rend le test plus complet.
