## 개요

SBTI 테스트는 재미 중심으로 설계된 SBTI 성격 테스트입니다. 이 저장소는 다국어 정적 SBTI 사이트 구현을 제공하며, 영어 `README.md`를 기본 프로젝트 문서로 사용합니다.

## SBTI 테스트란?

SBTI는 `Silly Big Personality Test`의 약자입니다. SBTI 성격 테스트는 질문에 답하고 결과를 보는 익숙한 흐름을 유지하지만, 전체 톤은 패러디, 자기풍자, 유머, 공유성에 맞춰져 있습니다.

- SBTI 테스트는 무료입니다
- SBTI 성격 테스트는 오락 목적입니다
- 결과를 캡처하고 공유하기 좋습니다
- 의학적 또는 심리학적 진단 도구가 아닙니다

## SBTI와 MBTI의 관계

SBTI 테스트는 MBTI의 유머러스한 패러디로 설명됩니다. MBTI에 익숙한 사람도 이해하기 쉽지만, SBTI 자체는 진지한 성격 이론으로 다루지 않습니다.

## SBTI 성격 테스트의 15개 차원

SBTI 성격 테스트는 15개 차원을 사용합니다. 이 차원들은 다음 5개 그룹으로 정리됩니다.

- 자기 모델
- 감정 모델
- 태도 모델
- 행동 구동 모델
- 사회 모델

그래서 SBTI는 농담 중심의 테스트이면서도, 결과를 여러 관점으로 정리한 뒤 최종 유형을 보여줍니다.

## FAQ

### SBTI란 무엇인가요?

SBTI는 결과를 가볍게 즐기고 공유하기 위한 유머 중심의 SBTI 성격 테스트입니다.

### SBTI와 MBTI는 어떻게 다른가요?

MBTI가 비교 기준이라면, SBTI는 더 가볍고 더 풍자적인 대안입니다. 재미가 핵심이며 과학적 정확성을 약속하지 않습니다.

### 왜 SBTI에는 15개 차원이 있나요?

결과를 여러 관점에서 정리하기 위해서입니다. 15개 차원이 최종 유형에 조금 더 입체적인 느낌을 줍니다.
