## 이 가이드 사용법

1. 먼저 원하는 성격 유형을 정한 뒤 아래에서 찾습니다.
2. 테스트를 열고 “테스트 시작”을 누릅니다.
3. 문항 순서는 매번 섞이므로, 문장 내용으로 어떤 문제인지 확인하세요.
4. 원하는 성격의 답안 카드에서 그 문항을 찾고 “선택” 열에 적힌 글자대로 누르면 됩니다.
5. 선택지는 A / B / C 입니다. A = 첫 번째, B = 두 번째, C = 세 번째입니다.
6. 일반 문항 30개와 특수 문항 1개를 모두 끝내면 “제출하고 결과 보기”를 누르세요.

중요: 테스트 중에는 취미를 묻는 특수 문항이 4지선다로 랜덤 등장합니다. DRUNK를 노리지 않는다면 “음주”는 절대 고르지 마세요. 나머지 세 개는 아무거나 괜찮습니다.

- 같은 차원에 속한 두 문항에는 일부러 반대 답을 주십시오. 예를 들어 한쪽은 A, 다른 한쪽은 C로 두면 점수가 중간에 떨어져 특정 프로토타입과 정확히 맞기 어려워집니다.
- 서로 다른 차원끼리의 패턴도 최대한 충돌하도록 만드는 것이 좋습니다.
